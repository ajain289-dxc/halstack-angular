Name Default Description options: object[] [] An array of objects representing the each toggle. Each object has the following properties:
  • value: Toggle's inner value.
  • label: String with the toggle's display value.
  • iconSrc: URL of the icon that will be placed in the toggle. If there is an iconSrc the label won't be shown.
value: any | any[] Value(s) of the toggle(s) that are toggled. If undefined, the component will be uncontrolled and the value will be managed internally by the component. multiple: boolean false If false only one toggle can be toggled. disabled: boolean false If true, the component will be disabled. onChange: EventEmitter This function will be called when the user changes the state of any toggle. The new values in an array will be passed as a parameter. margin: any (string | object) Size of the margin to be applied to the component ('xxsmall' | 'xsmall' | 'small' | 'medium' | 'large' | 'xlarge' | 'xxlarge'). You can pass an object with 'top', 'bottom', 'left' and 'right' properties in order to specify different padding sizes.
