Name Default Description currentPage: number 1 Number of the current selected page. itemsPerPage: number 5 Number of items per page. totalItems: number 1 Total number of items in the pages. paginationActions: string[] Pagination actions to be rendered ['prev', 'next', 'first','last']. In case the property is not defined all actions will be rendered. nextFunction: EventEmitter This function will be called when the user clicks the button to go to the next page. prevFunction: EventEmitter This function will be called when the user clicks the button to go to the previous page. firstFunction: EventEmitter This function will be called when the user clicks the button to go to the first page. lastFunction: EventEmitter This function will be called when the user clicks the button to go to the last page.
