Name Default Description currentPage: number 1 Number of the current selected page. itemsPerPage: number 5 Number of items per page. showGoToPage: boolean false Show page navigation select. itemsPerPageOptions: number[] [] An array of objects representing the selectable options of the items per page input. totalItems: number 1 Total number of items in the pages. paginationActions: string[] Pagination actions to be rendered ['prev', 'next', 'first','last']. In case the property is not defined all actions will be rendered. onGoToPage: EventEmitter This event will emit the new page when the user interacts with the page navigation buttons or select. itemsPerPageFunction: EventEmitter This function will be called when the user picks an option for the itemsPerPage select. The value selected will be passed as a parameter.
